Defines alpha-blend flags. 

If you're familiar with OpenGL or D3D, these flags aren't new to you. They define $\ast$how$\ast$ the final color value of a pixel is computed, basing on the previous color at that pixel and the new color value from the material. The blend formula is: 

\begin{Code}\begin{verbatim}    SourceColor * SourceBlend + DestColor * DestBlend
\end{verbatim}
\end{Code}

 where DestColor is the previous color in the framebuffer at this position and SourceColor is the material color before the transparency calculation.\par
 This corresponds to the AI\_\-MATKEY\_\-BLEND\_\-FUNC property. \begin{Desc}
