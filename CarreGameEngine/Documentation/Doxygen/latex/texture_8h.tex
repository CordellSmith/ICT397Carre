\subsection{Detailed Description}
Defines texture helper structures for the library. 

Used for file formats which embed their textures into the model file. Supported are both normal textures, which are stored as uncompressed pixels, and \char`\"{}compressed\char`\"{} textures, which are stored in a file format such as PNG or TGA.
